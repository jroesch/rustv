rustv

Rustv is a tool for managing multiple versions of the compiler toolchain.

Rustv is heavily inspired by rbenv and tools like it. I got use to tools for managing multiple installations of a language. I was frustrated by this missing functionality when juggling multiple compiler versions to either test library code or my hacking at the compiler.

Use it to install a binaries by version name: 
 $ rustv install 0.11.0 
 $ rustv install nightly
Or by source: 
 $ rustv install --source 0.11.0
Or by a SHA: 
 $ rustv install --sha 2e385817926f6f914fbff482aab3a8b627e7feee
Set a global version: 
 $ rustv global nightly
Set a local version: 
 $ rustv local 0.11.0
and more!
